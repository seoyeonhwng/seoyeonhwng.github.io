글또 4기 활동이 끝났다. 약 6개월동안 [글또 4기 다짐글](https://seoyeonhwng.github.io/2020/03/01/%EA%B8%80%EB%98%90-4%EA%B8%B0-%EB%8B%A4%EC%A7%90%EA%B8%80/)을 포함하여 
11개의 글을 작성했다. 마지막 글로 글또 활동을 되돌아 보는 회고록을 작성하려 한다.

## 어떤 글을 작성 했나?
이번 글또 활동을 통해 회사 일을 하며 알게된 **AWS 서비스를 정리**하는 것이 첫번째 목표였다. 그래서 작성한 총 11개의 글 중 대부분은 AWS 관련 글이다. 정리한 서비스는 다음과 같다.

- [Lambda](https://seoyeonhwng.github.io/2020/03/15/AWS-Lambda%EB%9E%80/) + 
[Lambda 배포 스크립트](https://seoyeonhwng.github.io/2020/03/22/AWS-Lambda-%EB%B0%B0%ED%8F%AC-%ED%8C%A8%ED%82%A4%EC%A7%95/) + 
[Lambda Layer 만들기](https://seoyeonhwng.github.io/2020/04/26/docker%EB%A1%9C-lambda-layer-%EB%A7%8C%EB%93%A4%EA%B8%B0/)
- [S3](https://seoyeonhwng.github.io/2020/04/11/AWS-S3%EB%9E%80/)
- [RDS](https://seoyeonhwng.github.io/2020/05/10/AWS-RDS-python%EC%9C%BC%EB%A1%9C-%EC%97%B0%EB%8F%99%ED%95%98%EA%B8%B0/)
- [EC2](https://seoyeonhwng.github.io/2020/08/01/AWS-EC2%EB%9E%80-%EB%AC%B4%EC%97%87%EC%9D%B8%EA%B0%80/)
- [AWS 서비스 모니터링](https://seoyeonhwng.github.io/2020/06/21/AWS-%EC%84%9C%EB%B9%84%EC%8A%A4-%EB%AA%A8%EB%8B%88%ED%84%B0%EB%A7%81-%EA%B8%B0%EB%8A%A5-%EB%A7%8C%EB%93%A4%EA%B8%B0/)

AWS 정리글 외에도 두번째 목표인 회고록도 작성하였다.
- [OKR 회고록](https://seoyeonhwng.github.io/2020/04/06/OKR%EA%B3%BC-OKR-%ED%9A%8C%EA%B3%A0/)
- [스타트업 인턴 회고록](https://seoyeonhwng.github.io/2020/07/19/%EC%8A%A4%ED%83%80%ED%8A%B8%EC%97%85-%EC%9D%B8%ED%84%B4-%ED%9A%8C%EA%B3%A0%EB%A1%9D/)

계획에는 없었지만 [screen 정리글](https://seoyeonhwng.github.io/2020/06/07/screen-command/)도 작성하였다. 우연히 알게되었는데 유용해서 까먹지 않기 위해 바로 정리했다. 
시간이 남으면 도커도 정리하고 싶었으나 시간이 남지 않았다 :(

## 느낀점
글또를 지원할때만 해도 2주마다 글 1개 작성이 어렵지 않을거라고 생각했다. 막상 해보니 생각보다 2주는 빨리 돌아왔고, 글 1개 작성은 벅차고 힘들었다. 
대충 글을 쓰고 싶지 않은 마음에 평균 3일에 거쳐 글을 쓰고 다시 읽어보고 다듬었다. 글또를 하지 않았다면 지금까지 꾸준히 블로그 활동을 하지 않았을 것이다. 내 자신을 믿지 않고 다음 기수에도 지원할 예정이다. 꼭…!

AWS를 하나도 모르는 사람도 내 글을 읽고 쉽게 이해할 수 있게 정리하고 싶었다. 하지만 쉽게 이해할 수 있게 정리하기 위해서 나는 더 많은 것을 자세히 알고 있어야 한다는 것을 글을 쓰면서 깨달았다. 
글을 적기 위해 AWS 공식 문서도 다시 읽어보면서 공부할 수 있는 좋은 시간이었다. 글또 분들이 남겨주신 피드백도 많은 도움이 되었다.

마지막으로 보증금 10만원 지키기를 성공했다! 무척 뿌듯하다.

## 다음 기수에는?
다짐글에서 목표했던 AWS 서비스중에서 아직 정리하지 못한 서비스들이 많다. 다음 기수에서 이어서 정리하고 싶다.
- DynamoDB
- API Gateway
- 두 개 이상의 서비스를 같이 ex) Lambda + API Gateway, …

하반기에는 막학기를 다니며 취업 준비를 할 예정이다. 학교 수업(기계학습, 딥러닝, 클라우드 컴퓨팅)에서 배운 내용이나 알고리즘, 코딩 테스트 관련 내용도 작성할 계획이다.

## 마치며
글또를 하면서 중간에 포기하고 싶었던 적이 많았는데 포기하지 않고 글또 4기 회고록으로 마무리하는 내자신이 기특하고 대견하다!
